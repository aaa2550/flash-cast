
## 技术实现

### 数据结构
```typescript
interface TemplateVideo {
  id: string;           // 视频ID
  title: string;        // 视频标题
  description: string;  // 视频描述
  thumbnailUrl: string; // 缩略图URL
  videoUrl: string;     // 视频URL
  duration: number;     // 时长（秒）
  category: string;     // 分类
  isPopular?: boolean;  // 是否热门
}
```
